Traitor Chess

How To Play

Welcome to Traitor Chess! This game puts a little twist on the traditional game of chess.

In Traitor Chess, after a set number of moves, one random piece from both white and black side will change color.

The "Impending Doom" countdown tells you when this color change will take place.

I'm currently trying to implement a "go back" button so that you can go back a turn if you made a mistake.

Currently the "check/mate" system is a work in progress but so far I have a temporary check system that restricts the king's moves.

However, there is pawn swap implemented for when you reach the end of the board as a pawn!

Hope you have fun playing!
